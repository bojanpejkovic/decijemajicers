DOCUMENTATION

Prepration

Before you proceed with HTML template installation, you need to prepare the following:

Software

We recommend that you download all the required software to edit the template. Softwares required are listed below:

  • You may need WinZip 9+ for Windows and Stuffit Expander 10+ Mac to extract zip file.
  • You should also use code editor, i. e. NetBeans IDE, Sublime Text, Notepad++, etc. for template files source code editing.
  • For file uploading to server, FTP manager is also required like File Manager at your hosting control panel or desktop applications, such as FileZilla, CuteFTP, etc.
Hosting

You can just run index.html file from the site folder directly after extracting zip file. Or you can upload template to hosting server if you need to make it live.

Template Upload

After extracting zip file you need to upload all the inner content of site folder to the root of your website hosting. (Just to make it live)

Note: The root directory of a site depends on the internal structure of your hosting server. If you upload the folder itself on hosting, the root of your site will be http://your_domain_name/folder_name. You need to upload inner content of zip file if you want url to be http://your_domain_name/
Warning: Do not upload the documentation folder to your hosting server. You only need to upload inner content of site folder.
