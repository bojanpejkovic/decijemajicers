Customization

Changing the Logo

In HTML template, you can use either image or text logo.

Attention: This is HTML template so you have to change logo manually in every file.
Changing the Logo

Div with class header-logo contains the logo so you need to change the image source path there or you can type text in the place of image as logo.

<div class="header-logo">
    <a href="index.html"><img class="img-fluid" src="assets/images/logo.svg" alt="" /></a>
</div>
Changing footer Logo

Div with class footer-logo contains the logo so you need to change the image source path there or you can type text in the place of image as logo.

<div class="footer-logo">
   <a href="index.html"><img src="assets/images/Footer-Logo.svg" alt="" /></a>
</div>
Adding a new menu item

Add new li element same as below example in the ul that have these classes menu-section mg-menu-outer. You can find this ul in the header tag.

<ul class="menu-section mg-menu-outer">
   <li class="menu-item"><a href="contact-us.html">Contact</a></li>
</ul>
Making menu item active

Add active class in li to make menu item active.

<ul>
   <li class="menu-item active"><a href="blog.html">Blog</a></li>
</ul>
Adding a submenu

Add ul same as below example in li (menu item) and add this class menu-item-has-children to the same li.

<li class="menu-item-has-children">
   <a href="javascript:void(0);">Pages <i class="fa fa-caret-down" aria-hidden="true"></i></a>
   <div class="mg-sub-menu-outer">
      <div class="menu-subs">
         <ul>
            <li><a href="collection.html">New submenu</a></li>
         </ul>
      </div>
   </div>
</li>
Change slider image

Change background-image url in style attribute on the div having class name mg-home-banner-outer.

<div class="mg-home-banner-outer d-flex" style="background-image: url('assets/images/banners/home1-b1.jpg');">
   ...
</div>
