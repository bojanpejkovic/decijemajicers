Structure

This section have complete information of file structure:

Package Structure

After extracting zip file, you'll see the following file structure:

documentation This folder contains documentation for this HTML template.

site This folder contains the HTML template.

site/assets This folder contains sub-folders for CSS, JS, images, and fonts.

site/assets/css All the CSS files are located in this folder.

site/assets/js Libraries and common JS files are located in this folder.

site/assets/js/pages JS files for specific pages like home page, product page, collections page and all others are located in this folder.

site/assets/images/banners Banner images for all three home pages are located in this folder.

site/assets/images/blog Blog images are located in this folder.

site/assets/images All other images are located in this folder.

site/assets/fonts All fonts are located in this folder.

HTML Structure

HTML page structure is built on the following principle:

<html lang="en">
   <head>
      <meta charset="UTF-8">
      <meta name="viewport" content="width=device-width, initial-scale=1.0">
      <!--Styles & Scripts-->
   </head>
   <body>
      <!--Page Header-->
      <header>
         ...
      </header>

      <!--Page Sections-->
      <section>
         ...
      </section>
      
      <!--Page Footer-->
      <footer>
         ...
      </footer>

      <!--JavaScripts-->
      <script src="assets/js/mg-menu.js"></script>
      <script src="assets/js/mg-common.js"></script>
   </body>
</html>
CSS Structure
  • site/assets/css/style.css all style is written in this file.
  • site/assets/css/responsive.css all responsive style is written in this file.
  • site/assets/css/mg-menu.css all menu style is written in this file.
  • site/assets/css/mg-animations.css all animation style is written in this file.
JS Structure
  • Script for the menu is written in site/assets/js/mg-menu.js file.
  • Common scripts are written in site/assets/js/mg-common.js file.
  • Script for specific page like product page is written in site/assets/js/pages/mg-product.js file.
